# Syntecxhub_LandingPage_Eduflow
## What is Eduflow?
EduFlow is a modern UI/UX landing page design for an online learning platform, showcasing a clean layout, interactive features, and a professional design workflow, created entirely in Figma.
## Features of Eduflow
 - Expert-Led Courses
 - Flexible Learning
 - Progress Tracking
 - Verified Certifications
## Design Layout
The EduFlow landing page is designed with a clear and intuitive structure to ensure a smooth user experience. The layout follows a top-down flow, guiding users through information in a logical and engaging manner.
### Header & Navigation
Includes the EduFlow logo, primary navigation links Home, Features, Pricing, About and a clear call-to-action button for quick access.
### Hero Panel
Features a strong headline, short introductory description, and a prominent app store links to immediately capture user attention to download mobile app.
### Features Panel
Highlights the core offerings of EduFlow such as courses, flexibility options, progress tracking and certifications
### Pricing Panel
Features platform's tiers categorized into free and paid features, presented with concise descriptions and visual icons for easy understanding.
### Testimonials Panel
Showcases feedback from students to build credibility and trust, reinforcing the value and effectiveness of the platform.
### About Area
Provides a brief overview of EduFlow, explaining its mission, target audience, and the benefits it offers to learners.
### Call-to-Action & Dialog Box
Includes an interactive CTA button that opens a dialog box mockup, demonstrating how users can book or enroll in courses.
### Footer Section
Contains essential information such as copyright details and supporting social media links
